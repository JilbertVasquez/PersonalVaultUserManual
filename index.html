Vault

Abstract

Vault is a simple and intuitive desktop application developed in Java using the NetBeans IDE, designed to help users securely manage their various online accounts. The primary function of Vault is for an admin to create initial accounts for accessing the Vault system. Once these accounts are created by the admin, users can log in and add accounts such as their Gmail, providing details like the username, site name, password, and additional info if desired. The application features a user-friendly interface where users can view all their stored accounts in a table format. Moreover, Vault offers functionalities to edit account details, when necessary, delete accounts, and perform searches to locate specific accounts with ease. This project aims to provide a secure and efficient solution for account management, ensuring that users can keep track of their account information with ease.

User

User Login

User will log in to their account by entering their username and password to use the system.

Add Account

After logging in, the user can add any account details such as the name of the site or app, their username, Gmail/email used in the site/app, the password, and additional info if needed. once they are done, they can now save the details by hitting the save button.

Accounts

The user can view all your added account info. by going to the account section, you can refresh the table to display all the updated account info. You can also search specific account information by just searching keywords in the search bar. Also, you can select a particular account, edit it, or delete it.

Edit Account

User can edit their account whenever they want. This will help to keep their account information updated if they update their social media accounts.

History

User have a history section to track their overall doings in their account. they can view when they added the account information, if they deleted it, or modify it.

Administrator

Admin Login

Admin are the ones who can add accounts for users. This will help to ensure that every user's data is organized in a database. Although the interface is the same as the user login, the system will automatically identify the it is the user or the admin who is logging in.

Adding of User Account

Admin has the control to add users by providing first name, last name, the user username, and password that will help them to log in and use the system.
